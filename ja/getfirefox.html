フォクすけの壁紙メーカーはこんなことができるよ！

フォクすけの壁紙メーカーは、日本発の Firefox のマスコットキャラクター「フォクすけ」をテーマにした壁紙を Firefox 上で簡単に作ることができるツールです。「パソコンのデスクトップ壁紙」や「携帯電話の待受画面」の他にも、Firefox のウィンドウデザインを着せ替えられる「Personas テーマ」のデザインも作ることができます。

このツールは、「SVG (エス・ブイ・ジー)」という Web の標準技術を使って作られており、プラグインをインストールしなくても、ブラウザ上で、画像を自由に加工するなど色々な面白いことができる一歩先の Web 技術の世界を、壁紙作りを通して皆さんに体験していただくことができます。

使い方は簡単！ まずは、作りたい壁紙の種類を選んだら、好きなポーズのフォクすけや背景を選択して、フォクすけの大きさを変えたり、位置を移動させたりして、自分だけのオリジナルデザインを作成してみましょう。自分のデザインが決まったら、あとは設定するだけ！
さぁ、今すぐ自分だけのオリジナル壁紙を作って、進化する Web の世界を味わってください。

SVG とは？

イラストや図形などの「形」を記憶する画像形式のひとつです。
次世代の Web 標準技術として、とても注目されています。

よく使われる画像データには、「JPEG (ジェイペグ)」や「PNG (ピング)」といったものがありますが、これらは、一度作ってしまったデータをきれいに拡大縮小したり、画像の中の一部分だけを編集したりすることはできません。「小さい画像を拡大したら、なんか線がギザギザに...」なんてこともありますよね。

SVG (エス・ブイ・ジー) という技術を使えば、このツールのようにどんな大きさに拡大縮小しても常になめらかな線で表示することができ、イラストや図表の位置や大きさを自由に変更できます。Web 標準技術を大切にし、SVG の可能性に注目してきた Mozilla は、5 年前からずっとこの技術をサポートしてきています。
