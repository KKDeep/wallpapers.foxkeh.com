フォクすけの壁紙メーカーは、最新版の Firefox で使ってね！

/

あなたが今お使いのブラウザ (インターネット閲覧ソフト) は、Web 標準技術である SVG に対応していません。フォクすけの壁紙メーカーをお使いいただくには、SVG に対応したブラウザが必要です。

ぜひお使いのパソコンに Firefox をインストールしてからこのページをご覧ください。

▶ Firefox ってなに？

壁紙メーカーのつかいかた

まずはフォクすけを選ぼう！

最初の画面では、今月のオススメ壁紙デザインが表示されているので、まずはフォクすけを変えてみよう。画面右側にあるカスタムパレットの中から、好きなフォクすけを選んでみてね。拡大縮小バーを使ってフォクすけの大きさを変更したり、透過させたりすることも簡単！ 画面上のフォクすけをクリックすると選択されるので、そのままドラッグして好きな場所に置くこともできるよ。

背景を変えてみよう！

次に背景を変えてみよう！ 画面上の背景部分にポインタを当ててクリックすると、右側のカスタムパレット上に色々な背景デザインが表示されるよ。好きな背景デザインをクリックして、壁紙の背景画像を変えてみよう。

カレンダーや Firefox ロゴの大きさもカスタマイズ！

デスクトック壁紙のカレンダー部分や Firefox ロゴマークもクリックすると、大きさや位置などがカスタマイズできるよ！

実際に壁紙として使ってみよう！

これだ！と思う壁紙ができたら、「PREVIEW」ボタンをクリックしてデザインを確認しよう。OKだったらそのままデスクトップ背景に設定できるよ。携帯待ち受け画面の場合はメールでデザインを送信することも可能。Personas テーマの場合は、プレビュー画面で「WEAR IT」をクリックしてから、テーマのインストールを許可すると、Firefox を着せ替えられるよ。

今後、新しい素材も追加予定！

フォクすけの壁紙メーカーでは、今後、新しい壁紙素材も追加される予定！お楽しみに！

 

CUSTOMIZE

SVG とは？
SVG 使ってます！
フォクすけの壁紙メーカーでは「SVG (エスブイジー)」という Web 標準技術を使って、プラグインなどをインストールしなくてもブラウザだけで簡単に自分だけのオリジナル壁紙を作れます。
SVG ってなに？
SVG は、イラストや図形の形と位置を記憶している画像形式のひとつで、次世代の Web 標準技術として注目されています。形や位置が記憶されているので、画像の一部分だけを移動させたり、大きさを自由に変更することができ、拡大しても常になめらかな線で表示させることができます。スクリプトを使って簡単に加工できるのも大きな特徴です。フォクすけの壁紙メーカーでは、この技術を使って、素材 (フォクすけ、背景、カレンダー、Firefox ロゴ) を自分の好きなようにカスタマイズできます。
Mozilla の SVG への取り組み
Web 標準技術を大切にしている Mozilla では、既に 5 年前から Firefox 上で SVG 画像を表示できるようにしています。さらに最近では、これまで SVG でしか利用できなかったぼかしなどの特殊な効果を、SVG 以外でも利用できるようにしました。また、SMIL (SVG アニメーション) という技術などの導入にも積極的に取り組んでいます。
SVG の可能性
SVG 画像は、ブラウザ上でスクリプトを用いて、コントロールすることができます。たとえば、地図を拡大、縮小するようなアプリケーションでも、画像の表示は常になめらかです。地名や建物名を示す文字、コンビニや信号を表す画像を埋めむこともできます。また、アニメーションやフィルタの操作も簡単にできます。SVG によって、表現の幅が今まで以上に広がることでしょう。このツールでは、SVG の可能性の一部をお見せします。

その他の FAQ
対応ブラウザは？
Firefox 3.6、Opera 10.53、Safari 4/5、Chrome 5 で動作確認を行っています。Internet Explorer は SVG に対応していないため動作しません。
自分の画面サイズが正しく選択されません。
通常は、お使いのパソコンの画面サイズが自動的に選ばれています。だたし、ページを拡大表示している場合は、Ctrl+0 キーを押してサイズから サイズ選択ページ を再読み込みしてください。詳しくは Bug 547469 (英語) をご覧ください。
